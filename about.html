About PWS 2025

Procyon WebServer is a fast webserver with SLS/LSP and PHP support. It is open source and free to use. It is designed to be fast and efficient, and is perfect for small to medium websites. It is easy to install and configure, and is compatible with most web browsers. Procyon WebServer is the perfect choice for anyone looking to create a professional website quickly and easily.

  • SLS/LSP - Server Logic Script/Linea Server Pages Support.
  • PHP - PHP Hypertext Preprocessor Support (needs PHP binaries in server folder)
